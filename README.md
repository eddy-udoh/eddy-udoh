## 🔬 What I'm building right now

**[MedSpeak](https://github.com/eddy-udoh/fyp-yoruba-english-s2st)** — my final year project. A two-way Yoruba ↔ English speech-to-speech translator for Nigerian clinics: a patient describes their symptoms in Yoruba, the doctor hears it in English seconds later, and the reply comes back the same way.

Under the hood, I fine-tuned Whisper for Yoruba speech recognition, adapted MarianMT for medical vocabulary, and used VITS for speech synthesis — all served through a Flask API with a React frontend. Building for a low-resource, tonal language has been the hardest part of the project. Yoruba's diacritics have taught me more patience than any compiler error ever did.
